# 5 Fold Life

## What is here

- `netlify-export.zip`: ready to upload directly to Netlify
- `source/`: the editable project files

## Source contents

- `source/frontend/`: static site publish folder
- `source/backend/`: backend files kept with the project
- `source/netlify.toml`: Netlify config for repo-based deploys
- `source/README_NETLIFY.md`: Netlify deployment notes

## Simplest path

If you want the fastest deploy, use `netlify-export.zip`.

If you want to keep editing the project, work inside `source/`.

## Notes

- The results screen now uses a generated profile emblem instead of heavyweight mascot PNG files, which keeps the editable workspace and export much smaller.
